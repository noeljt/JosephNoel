Gentle Jungle

Complete website redesign

The Gentle Jungle is a family owned pet store in Meriden, CT. Their website was previously created by the owner's daughter and hosted by Wix.com, costing the business over $100 per month. It was not responsive or mobile-friendly and had poor search engine optimizations.

The new website was built from a bootstrap template in an agile fashion with weekly meetings with the business owner. The redesigned website is completely responsive, including a custom made instagram feed, new logo, and color palette. Hosted on GoDaddy, the new website cost $5 a month, a 95% reduction in cost.

Tweet n Load

MEAN stack and Twitter API

This app was created during an IT class at RPI, but I added the challenge of using an unfamiliar frontend library (Angular Material instead of Bootstrap). The app can load tweets given user inputted amounts and optional keyword, display them on screen, export the database to a file, visualize some interesting data, and reset the database.

The app runs on a MEAN stack, with a mongoDB instance hosted by mLab. While the use of socket.io allows multiple users to connect and use this project at the same time, it all shares one database, so if you get some extra tweets or the database is reset, don't be too sad.
